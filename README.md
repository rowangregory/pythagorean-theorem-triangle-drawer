## How can I install this?
Download the latest version of Python [https://www.python.org/downloads/] (here), and then download this as a ZIP file and then unzip it. Activate the IDLE program, click "Open…" and find "main.py" and click on it. Note that it asks you directly in the shell, because I don't know how to make a GUI yet.
